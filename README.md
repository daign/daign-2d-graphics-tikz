# daign-2d-graphics-tikz

[![NPM package][npm]][npm-url]

A TikZ renderer for the [daign-2d-graphics][library-url] library

## Installation

```sh
npm install @daign/2d-graphics-tikz --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
